# Automatic Water Level Controller

## Project Description
This project automatically controls a water pump using Arduino.

## Components
- Arduino Uno
- Water Level Sensors (2)
- Relay Module
- Buzzer
- Green LED
- Red LED

## Pin Configuration
Low Sensor - D2
High Sensor - D3
Relay - D6
Green LED - D7
Red LED - D8
Buzzer - D9

## Working
If water level is LOW → Pump ON
If water level is HIGH → Pump OFF + Buzzer Alert
